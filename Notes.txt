# How to

## Get where the CANUSB is connected

use the terminal blocked command to monitor system events:

```bash
sudo dmesg -wL
```
Once the can device connected:

```bash
[81442.228943] can0: slcan on ttyUSB7.
```

## acticate slcand daemon

```bash
sudo slcand -o -c -f -s8 /dev/ttyUSB7 slcan0

sudo ifconfig slcan0 up 
```
